# Investment Behavior Data Analysis

## Project Overview
This project analyzes an investment behavior dataset to understand investor preferences, objectives, and expectations. The analysis is performed using Python, Pandas, and Matplotlib in Jupyter Notebook.

## Dataset
The dataset contains information about:
- Gender
- Age
- Investment avenues
- Savings objectives
- Investment duration
- Sources of investment information
- Expected returns

## Analysis Performed

1. Gender Distribution of Investors
2. Statistical Analysis of Numerical Columns
3. Most Preferred Investment Avenue
4. Reasons for Investment
5. Savings Objectives Analysis
6. Common Information Sources
7. Investment Duration Analysis
8. Preferred Investment Period
9. Age Distribution of Investors

## Key Insights

- Mutual Funds are the most preferred investment avenue.
- Capital Appreciation is the most common savings objective.
- Retirement planning is the major reason for investing.
- Financial Consultants are the most common source of investment information.
- Most investors prefer investment durations of 3–5 years.
